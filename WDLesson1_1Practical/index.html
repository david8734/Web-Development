Web Technologies

HTML

HTML (HyperText Markup Language) is the standard language used to create and design webpages. It structures content using elements like headings, paragraphs, and links, allowing browsers to display text, images, and interactive features. HTML documents are made up of tags, which define how content is presented on the web.

CS5

CS5 (Creative Suite 5) is a software package developed by Adobe that includes a collection of design and development tools like Photoshop, Illustrator, and Dreamweaver. It was released in 2010 and is used by professionals for graphic design, web development, video editing, and more. CS5 was later succeeded by Adobe's subscription-based Creative Cloud (CC) suite.

JavaScript

JavaScript is a programming language commonly used to create interactive and dynamic content on websites. It allows developers to control elements like animations, form validation, and user interactions without needing to reload the page. JavaScript runs in the browser and is an essential part of front-end web development, often working alongside HTML and CSS.
